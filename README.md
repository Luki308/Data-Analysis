# Data Analysis & Visualizations

This repository presents a collection of projects focused on providing appropriate visualizations of data on interactive dashboards. It acts as a **navigation hub** that explains briefly each project and links to its dedicated implementation repository.

---

## Featured projects

### Spotify Shared Music Dashboard (Python)

**Problem:**
How can shared music listening patterns between users or groups be visualized in an
intuitive and exploratory way?

**Outcome:**
A user-friendly interactive flow chart that reveals overlap in music preferences and listening habits across individuals or groups.

➡️ **Repository:** [Spotify_Dashboard](https://github.com/Luki308/Spotify_Dashboard)

![spotify_image](https://github.com/Luki308/Spotify_Dashboard/blob/main/image.png?raw=true)

### Cryptocurrency & Stock Markets Dashboard (Python + Apache Big Data)

**Problem:**

How can real-time and historical market data be combined into a meaningful visualizations that drive decision making of traders?

**Outcome:**

An IaaC service that provided real-time crypto and stock market data on dashboard with predictions of a price for the next time window.

➡️ **Repository:** [CryptoStockPrediction](https://github.com/szuvarska/CryptoStockPrediction)

### NYC-Citibike Dashboard (R)

**Problem:**
How did the COVID-19 pandemic affect bicycle usage patterns in New York City, and how does bike traffic vary spatially across neighborhoods and time of day?

**Outcome:**
An interactive Shiny dashboard enabling users to explore temporal and spatial bike usage patterns in NYC, supporting intuitive comparison across years, locations, and time windows.

➡️ **Repository:** [NYC---bikes-analysis](https://github.com/Luki308/NYC---bikes-analysis)
